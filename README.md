# DocuMind AI 📄

An AI-powered technical documentation generator that automatically creates clean, structured documentation from your code.

## 🚀 Features
- Supports Python, JavaScript, TypeScript, Java, Go, SQL
- Generates Overview, Functions, and Usage Examples instantly
- Copy as Markdown or Download as .md file
- Powered by Groq AI (Llama 3.1)

## 🛠️ Tech Stack
- React.js
- Groq AI API (Llama 3.1)
- CSS3

## ⚙️ Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env` file with `REACT_APP_GROQ_KEY=your_key`
4. Run `npm start`
